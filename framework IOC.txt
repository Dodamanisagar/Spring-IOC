steps in framework:-
1.creat a main classs
2.build a path for required jar files
3.creat container inside main class and pass xml.file inside that (we used ApplicationContext container)
4.creat another class of which we have to creat object in side that generate setter and getter methods and override tostring method
5.creat xml.file----->tells container ,which objects we want to creat (xml.file must be inside src file)

main class:-
